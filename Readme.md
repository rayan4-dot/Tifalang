
# ⛰️ TifaLang (ⵜⵉⴼⴰⵍⴰⵏⴳ)

TifaLang is a professional, bytecode-powered programming language built for the **Tifinagh script**. It is a complete, self-contained ecosystem for modern Amazigh software engineering.

## 🚀 One-Liner Global Install

Install the TifaLang engine and ecosystem on any Mac or Linux machine with one command:

```bash
curl -fsSL https://raw.githubusercontent.com/rayan4-dot/Tifalang/main/get-tifa.sh | bash
```

## 💎 Ecosystem Power
TifaLang isn't just a command; it's a full platform. After installing, explore your new powers:

- **`tifa status`**: View your installed libraries and system integration.
- **`tifa help`**: View all Tifinagh-native commands.
- **`tifa new <name>`**: Start a new Amazigh software project.
- **`tifa run <file>`**: Execute your Tifinagh code instantly.

## 🛠️ System Integration
To integrate TifaLang with your favorite tools:

### VSCode Development
```bash
tifa setup-vscode
```
*Installs the Tifinagh syntax highlighter and developer tools.*

### Keyboard Support
```bash
tifa setup-keyboard
```
*Opens the Tifinagh keylayout for your operating system.*
